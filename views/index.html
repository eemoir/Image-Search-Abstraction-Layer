User Stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string by adding /api/imagesearch/parameters to the above URL.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings through /api/atest/imagesearch.
